# Heart Rate and SpO2 Monitoring System

This project measures heart rate (BPM) and oxygen saturation (SpO2) using the MAX30100 sensor and Arduino Uno.

## Components
- Arduino Uno
- MAX30100 Pulse Oximeter Sensor
- 16x2 LCD Display
- Breadboard and jumper wires

## Features
- Real-time BPM measurement
- SpO2 monitoring
- LCD display output

## Technologies
Arduino, Embedded C, I2C communication# heart-rate-spo2-monitor-arduino
